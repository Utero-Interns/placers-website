# Roadmap: Placers Website

## Overview

Transform the Placers billboard advertising marketplace from a frontend-only prototype into a fully functional platform. This journey involves connecting all existing pages to the backend API (63 endpoints), building out seller and admin dashboards, implementing notifications and wallet systems, and polishing UX for production launch.
